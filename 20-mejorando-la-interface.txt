

MEJORANDO LA INTERFACE 


utilizamos children para utilizar solo una vez el componente repetitivo <navbar/>



en el archivo app.js

importaremos el componente Layout desde layout.js

con la siguiente extructura:

layout.js:

import React from 'react'
import NavBar from './NavBar'
import logo from '../images/logo.svg'


const Layout = props => {
  return(
    <>
      <NavBar logo={logo}/>
       {props.children}
    </>
  )
}

export default Layout


usaremos {props.children} para anotar que todo lo que este enre las etiquetas <Layout></Layout> en el archivo App.js
es un children

app.js quedaria asi:

import React from 'react'
import BadgesNew from '../pages/BadgesNew'
import Badge from '../pages/Badge'
import {BrowserRouter as Router, Switch, Route} from 'react-router-dom'
import Layout from './layout'
import notFound from './notfound'


const App = () => {
  return(
    <Router>
      <Layout>
        <Switch>
          <Route exact path='/badges' component={Badge}/>
          <Route exact path='/badges/new' component={BadgesNew}/>
          <Route component={notFound}/>
        </Switch>  
      </Layout>
    </Router>
  )
}

export default App

la idea de esto es eliminar los elementos importados repetitivos en las paginas
por eso eliminamos el elememento navbar de las paginas badge y BadgesNew

eliminaremos los divs que engloban contenido dentro del jsx de los componentes
los remplazaremos por React.Fragment o sino por <></>

por ultimo crearemos un Router para las rutas que no coincidan con los paths

          <Route component={notFound}/>
